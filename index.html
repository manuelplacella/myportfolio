Latest works

Year Song Prediction

This project is the highlight of the "Data Analytics" course, aiming to predict the release year of songs through advanced supervised Machine Learning techniques, including neural networks and innovative models like TabNet and TabTransformer. Covering all phases from data acquisition to performance evaluation, it showcases the practical application of analytical theories in music analysis.

Details

Motion Deblurring

This Machine Learning course project addresses the challenge of artificially introduced image blur, simulating the effect of photos taken with shaky hands. By employing a neural network implemented with Keras, the project showcases the power of deep learning in enhancing image clarity, highlighting its practical applications in real-world image processing.

Details

AI for Olive Oil Authenticity

This Artificial Intelligence course project focuses on fighting olive oil fraud by authenticating the geographic origins of extra virgin and virgin olive oils (EVOO and VOO). It tackles food fraud by leveraging accessible AI and analytical methods, along with geographic markers, for origin verification, adhering to EU regulations. The aim is to boost consumer trust, increase market transparency, support informed decisions, and encourage fair competition among producers.

Details

PoS-Tagging

This study, a project from the NLP course, compares two advanced methodologies: LSTM and BERT Transformer, to evaluate their effectiveness in assigning grammatical categories. The goal is to explore how these technologies enhance the interpretation of natural language and promote the development of NLP applications, highlighting significant advancements in the field.

Details

Geo-referenced Photo Management System

This project is part of the "Context-Aware System" course and focuses on creating a software platform for managing georeferenced multimedia files, primarily photographic images. The main objective is to provide users with the ability to capture new multimedia content via mobile devices and access it easily through an intuitive web interface. Key features include advanced spatial data analysis for visualizing geographic information and creating clusters of geographic markers related to the images.

Details

RPA techniques for a questionnaire generation engine

This work is my graduation thesis project, which consists of designing and implementing an RPA software application in Java language, with a GUI, to automate various processes. Among the main automated operations are the execution of a questionnaire generation engine in XML format, the transformation of questionnaires into PDF format, the creation of a single PDF file containing all the questionnaires, the compilation of Excel spreadsheets, and the publication of data on a web server for the degree course.
